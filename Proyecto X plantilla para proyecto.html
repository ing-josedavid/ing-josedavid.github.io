Dashboards e informes interactivos

Incluir un dashboard es una excelente opción, pero no es obligatorio en tu portafolio de MVP.
Plataformas recomendadas:
• Tableau Public: fácil de usar y permite publicar dashboards online.
• Power BI Service (gratuito): útil para informes empresariales.
• Streamlit, Plotly Dash, R Shiny: para dashboards personalizados si
programas en Python o R.

Ejemplo:
Un mapa interactivo que filtra datos por ubicación, o un gráfico que muestra tendencias en tiempo real.
Este tipo de evidencia no solo muestra tus habilidades técnicas, sino también tu capacidad de contar
historias con datos de manera clara y visual (data storytelling).
