Movilidad urbana y productividad económica para American Development Bank | Bootcamp Data Analyst TripleTen (2026)

Descripción gral.: Se analizarón datos de TomTom Traffic Index y OECD Cities para evaluar la relación entre movilidad urbana y productividad económica en ciudades latinoamericanas. Se limpiaron, integraron y analizaron los datos para identificar oportunidades de inversión en infraestructura de transporte y generar información útil para la toma de decisiones.

Descripción

• Integré y analicé datasets de movilidad urbana (TomTom Traffic Index) y productividad económica (OECD Cities)
para generar información estratégica sobre inversiones en infraestructura.
• Diseñé reportes automatizados con Python reduciendo significativamente el tiempo de elaboración de análisis.
• Construí una base de datos limpia y estandarizada para futuros estudios de movilidad y desempeño económico.

Evidencias visuales

Comparación de ambas variables, para observar si existe alguna relación entre ellas, mediante gráfico de barras donde aparecen ambos indicadores.

Comparación de las ciudades ordenadas por su PIB per capita de mayor a menor para facilitar la visualización de si existe alguna relación entre ellas mediante gráfico de barras donde aparecen ambos indicadores.
